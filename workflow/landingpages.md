Buatlah Landing Pages Untuk Memperkenalkan Sebuah Website VERA yang akan digunakan untuk survei penelitian di matakuliah UI UX

Penjelasan Vera AI:
Sebuah product SAAS Kasir Berbasis kecerdasan buatan dimana ada pelayanan jual beli dilakukan oleh asisten virtual dengan karakter 3d dengan AI seperti Vtuber. Produk ini bisa di beli/sewa oleh para pelaku UMKM F&B seperti cafe atau restoran

Fitur-Fitur:
- VERA AI (Asisten Kasir) dengan integrasi pembayaran QRIS
- Dashboard Realtime Pesanan Yang Masuk
- Kelola Menu
- Laporan Penjualan
- Konfigurasi Agent

Yang Harus Ada Di Landing PAGES:
- Hero Section
- About
- FAQ
- Ajakan Untuk Membantu Survei Pengembangan VERA
- CTA (Yang Akan Mengarah Ke Link Form Survei)

Design:
color-primary: #E86C1D
color-secondary : #9D4300 (warna kedua untuk hover atau button kedua)
color-tertiary: #FFFFFF (untuk background)
font: Plus Jakarta Sans

Aturan:
- Jangan Gunakan Emoji
- Gunakan Icon SVG Jika Butuh
- Jangan Gunakan Glassmorphism
- Gunakan Warna Aksen Super Taktis
- Gunakan Style Solid Rounded
- Gunakan bahasa yang lugas, spesifik, dan kasual.
- Gunakan bahasa indonesia
- Pemisahan Melalui Kontras Warna Dasar: Karena tidak pakai border atau shadow, pisahkan elemen menggunakan palet monokrom dengan kontras tinggi. Misalnya, gunakan latar belakang putih bersih untuk area konten atau card, dan warna abu-abu sangat terang untuk latar belakang utama (body). Perbedaan warna yang tegas namun minimalis ini akan membentuk struktur tanpa perlu garis.

Gunakan stack:
- Astro JS
- Tailwind CSS V4
- React (Jika Butuh)
